The joys of web development

Things have changed a lot since I first started messing around in this...

From tool availability to free resources, it's never been easier to get into!

I love web development and I think this course will further cement that!

I started working with HTML (on a super basic level) in middle school back in 2004. I used early Microsoft Word to "publish" basic webpages that I could share with my friends.

Fast forward about sixteen years, and things have gotten SO easy for people looking to pursue this as a career to find resources to learn. Resources like W3 Schools and Codecademy make it super easy to learn more about web development and how to use it effectively to make beautiful web designs and responsive pages.
